# Creative-development-101

## Description du projet

Ce projet est une expérience web interactive conçue pour évoquer et explorer différentes émotions à travers un design créatif avec des animations. Trois émotions y sont représentées, chacune utilisant des couleurs et des styles artistiques différents, illustrant la complexité et la diversité des émotions.

- **Frustration** : Représentée par un bouton qui fuis la souris, et une ambiance rouge et jaune (Feu, flammes).
- **Calme** : Une ambiance posée avec des bulles interactives, et une musique calme.
- **Joie** : Des rayons lumineux.

Chaque section de la page regroupe ces trois émotions.

## Instructions pour lancer le projet

1. **Cloner le dépôt** :

   ```bash
   git clone https://github.com/LucasAudoubert/Creative-development-101.git
   ```

2. **Naviguer dans le répertoire du projet** :

   ```bash
   cd Creative-development-101
   ```

3. **Installer les dépendances** :
   Assurez-vous que Node.js est installé, puis exécutez :

   ```bash
   npm install ou npm i
   ```

4. **Lancer le serveur de développement** :
   Démarrez le serveur de développement pour le projet localement :

   ```bash
   npm run dev
   ```

5. **Construire le projet** (Optionnel) :
   Pour créer une version de production, exécutez :

   ```bash
   npm run build
   ```

6. **Ouvrir dans le navigateur** :
   Une fois le serveur de développement lancé, ouvrez votre navigateur et accédez à :
   ```
   http://localhost:3000 ou l'url indiquer dans le terminale.
   ```
